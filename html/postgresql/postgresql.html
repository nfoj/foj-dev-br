PostgreSQL - Básico, Criando, Apagando e Inserindo

O que é Postgresql?

Postgresql é um sistema de gerenciamento de banco de dados relacional objeto-relacional (ORDBMS) de código aberto, altamente escalável, confiável e de alto desempenho.

Comandos Básicos


\h = ajuda!

\h create role = ajuda a criar uma regra
          
\? = lista comandos
          
\l = lista o banco de dados
          
\du = lista usuários
          
\c "banco de dados" = acessa o banco de dados especificado
          
\q = sair
          
\d = lista tabelas
\dS = lista tabelas do sistema
          
\! = terminal
exit > user sql = sai do terminal e volta para SQL
          
\! date = acessa o terminal
\! pwd = mostra o diretório atual

Criando um Banco de Dados


// #
CREATE DATABASE "nome" 
WITH
OWNER = postgres
ENCODING = 'UTF8'
CONNECTION LIMIT = -1;

-1: sem limite de conexão;

Deleta o Banco de Dados


DROP DATABASE "nome";

Tipos de Dados


- Números:
    integer (-1 ... 1)
    real (-1.0 ... 1.0)
    serial (+1.0 - Autoincrement)
    numeric (decimal types explict - .00, .000)  

- Texto:
    varchar (n) 
    char (n definition)
    text (no definition)

- Boolean:
    true
    false

- Tempo:
    date 'YYYY-MM-DD'
    time 'HH24:MM:SS'
    timestamp 'YYYY-MM-DD HH24:MM:SS'

Tipos em Uso


CREATE TABLE student (
    id SERIAL,
    name VARCHAR(255),
    cpf CHAR(11),
    observation TEXT,
    year INTEGER,
    money NUMERIC(10,2),
    height real,
    active BOOLEAN
    date_of_birth DATE
    hour TIME,
    registration TIMESTAMP        
);
                  
// NUMERIC(10,2): 12345678,90
