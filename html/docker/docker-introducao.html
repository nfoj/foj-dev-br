O que é Docker?

Docker é uma plataforma de desenvolvimento e distribuição de aplicativos que utiliza o conceito de containerização;

Um container é um ambiente isolado e portátil que contém tudo o que um aplicativo precisa para funcionar, incluindo código, bibliotecas e dependências;

Para que usar?

  • Executar um container Ubuntu;
  • Criar um ambiente PostgreSQL;
  • Usar uma imagem Docker do WordPress.
