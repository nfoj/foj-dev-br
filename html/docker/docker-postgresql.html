Docker - PostgreSQL

O que é Postgresql?

O PostgreSQL é uma ferramenta que atua como sistema de gerenciamento de bancos de dados relacionados.

Acesse:

  • PostgreSQL.

O que é da para fazer?

  • Armazenamento e recuperação de dados;
  • Gestão de dados;
  • Desenvolvimento de aplicações;
  • Analítica de dados;
  • Segurança de Dados.

Network

Criamos uma rede para poder conectar internet ao container.


 docker network create --driver bridge my-network
 docker network ls

Postgres

Os comandos a seguir devem seguir na mesma linha


 docker run 
  --name my-postgres 
  --network=my-network 
  -p 5433:5432 
  -e POSTGRES_PASSWORD=postgres 
  -d postgres
        

 docker inspect my-network

Postgres - pgAdmin


 docker run 
  --name my-pgadmin 
  --network=my-network 
  -p 15432:80 
  -e PGADMIN_DEFAULT_EMAIL=pgadmin@hotmail.com 
  -e PGADMIN_DEFAULT_PASSWORD=postgres 
  -d dpage/pgadmin4
        

 docker ps
