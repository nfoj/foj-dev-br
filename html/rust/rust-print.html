Rust {Print!}

Em Rust, a impressão de dados na tela é realizada principalmente através da macro println!, que permite formatar a saída de forma customizada utilizando placeholders. Os placeholders são representados por chaves {} e servem para inserir valores de variáveis ou expressões diretamente na string a ser impressa. Para tornar o código mais legível e compreensível, utilizamos comentários, que são linhas de texto ignoradas pelo compilador e servem para explicar o propósito de cada parte do código. Novembro - 01/2024

Comentários, print e placeholders

a. Comentários

  • // : usado para comentários com apenas 1 linha;
  • /* */ : usado para comentários que ocupem mais de uma linha.

Exemplo:

            
    fn main () {

        // Essa linha é um comentário


        /* 
            ...
            Esse comentário está num bloco!
            ...
        */

    }

b. Print

  • print! : imprime texto sem quebra de linha;
  • println! : imprime texto com quebra de linha;

Exemplo:

            
    fn main () {

        // Não quebra linha!
        print!("Hello, world!");
                

        //Quebra linha!
        println!("Olá, Maria!");
        println!("Olá, Carlos!");
        
    }

c. Quebra de Linha

  • \n : quebra linha

Exemplo:

            
    fn main () {
  
        println!("Qual o seu nome?\nMeu nome é Thays!");
        println!("1 + 1 =\n2");
        
    }

d. Placeholder

  • { }, {n}, {:n}, {:.n} : espaço reservado para texto ou código.

Exemplo:

            
    fn main () {

        //
        let name = "Alice";

        // {}
        println!("My name is {}", name);
        // My name is Alice


        // {...}
        println!("My name is {name}");
        // My name is Alice
