Comentários

  • Comentários de linha: iniciam com // e se estendem até o final da linha;
  • Comentários de bloco: são delimitados por /* e */, podendo abranger várias linhas.
    
      // Linha;
      // Comentários de linha;
    

      /* Bloco; */
      /*
      
        Comentários de bloco;
  
      */

Print

  • print!: imprime texto sem quebra de linha;
  • println!: imprime texto com quebra de linha;
    
      print!("Olá, Maria!");
      println!("Olá, Carlos!");

Quebra de linha

  • \n: deve ser usado para inserir uma quebra de linha.
    
        println!("O que é seu nome? \nMeu nome é Maria!");
        println!("1 + 1 = \n2");

Placeholders

  • {}: insere o valor do próximo argumento sem formatação;
  • {:.n}: formata números com "n" casas decimais.
