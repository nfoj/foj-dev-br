HackerRank - Tail of a Text File #1

Janeiro - 25/2025

Problem: In this challenge, we practice using the tail command to display the last n lines of a text file. Display the last 20 lines of an input file.

# Input

  • From fairest creatures we desire increase,
  • That thereby beauty's rose might never die,
  • ...

# Output

  • Feed'st thy light's flame with self-substantial fuel,
  • Making a famine where abundance lies,
  • ...

# Solution

            
  tail -n 20

# Solution_01

            
   while read line
  do
   echo "$line"
  done | tail -n 20
