HackerRank - Middle of a Text File

April - 25/2025

Problem: Display the lines (from line number 12 to 22, both inclusive) of a given text file.

# Input

  • From fairest creatures we desire increase,
  • That thereby beauty's rose might never die,
  • ...

# Output

  • And tender churl mak'st waste in niggarding:
  • Pity the world, or else this glutton be,
  • ...

# Solution

            
    head -n 22 | tail -n 11
