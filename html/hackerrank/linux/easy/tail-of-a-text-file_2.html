HackerRank - Tail of a Text File #2

April - 25/2025

Problem: In this challenge, we practice using the tail command to display the last n characters of a text file. Display the last 20 characters of an input file.

# Input

  • New York is a state in the Northeastern and Mid-Atlantic regions of the United States.
  • ...
  • Henry Hudson's 1609 voyage marked the beginning of European involvement with the area.

# Output

  • ent with the area.

# Solution

            
  tail -c 20

# Solution_01

            
  while read line
  do
    echo "$line"
  done | tail -c 20
