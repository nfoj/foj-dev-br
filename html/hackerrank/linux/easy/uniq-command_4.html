HackerRank - 'Uniq' command #4

January - 25/2025

Problem:

Given a text file, display only those lines which are not followed or preceded by identical replications.

# Input

  • A00
  • a0
  • 01
  • 01
  • ...

# Output

  • A00
  • a00
  • ...

# Solution

            
  uniq -u
