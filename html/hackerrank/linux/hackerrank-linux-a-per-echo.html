HackerRank - A Personalized Echo

Novembro - 22/2024

Problem:

Write a Bash script which accepts name as input and displays the greeting "Welcome (name)".

# Input

  • Dan

# Output

  • Welcome Dan

# Solution

            
    read name
    echo "Welcome $name"

# Comments

             
    # variable
    variable=Example

    # echo + text
    echo "Small test: $variable"

    # output
    Small test: Example

    --------------------------
                
    # read
    read text

    # echo
    echo "Welcome $text"

    # output
    Welcome text
  
    --------------------------

    \a: Alert (historically known as BEL). This generates the default alert sound.
    \b: Backspace.
    \c: Abort current output.
    \e: Escape.
    \f: Form feed.
    \n: Newline.
    \r: Carriage return.
    \t: Horizontal tab.
    \v: Vertical tab.
    \\: Backslash.
