HackerRank - Let's Echo

Novembro - 22/2024

Desafio:

Escreva um script bash que imprima a cadeia de caracteres "HELLO".

# Input

  • -

# Output

  • Hello

# Resposta

            
    echo Hello

# Resposta

            
    message='HELLO'
    echo "$message"

# Comentários

            
    # echo + texto
    echo Exemplo

    # saida
    Exemplo
    
    --------------------------
    
    # echo + " " > Não aparece
    echo "Exemplo"

    # saida
    Exemplo
    
    --------------------------
    
    # echo + " " > Aparece
    echo \"Exemplo\"

    # saida
    "Exemplo"

    
    --------------------------
    
    # variavel 
    variavel=Exemplo
    
    # echo + texto 
    echo "Pequeno teste: $variavel"

    # saida
    Pequeno teste: Exemplo

    --------------------------
    
    \a: Alerta (historicamente conhecido como BEL). Isso gera o som de alerta padrão.
    \b: Grava um caractere de backspace.
    \c: Abandona qualquer saída adicional.
    \e: Grava um caractere de escape.
    \f: Grava um caractere de alimentação de formulário.
    \n: Escreve uma nova linha.
    \r: Grava um retorno.
    \t: Escreve uma guia horizontal.
    \v: Grava uma guia vertical.
    \\: Escreve um caractere de barra invertida.
