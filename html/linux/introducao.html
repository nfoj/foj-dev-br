1. Introdução

1.1 Bios

- Bios - Basic input/output system:

  • Sistem boot process: Loads the operating system stored on the hard drive or other storage device;
  • POST (Power-On Self-Test): Checks RAM memory, processor, storage devices, and other components;
  • BIOS/UEFI setup: Allows you to set the time, date, boot order, and other options.

- UEFI: Unified extensible firmware interface (Bios + Upgrades):

  • Ethernet network configuration;
  • Fast boot;
  • UEFI Shell.
