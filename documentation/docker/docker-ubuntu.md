# Docker Ubuntu

## Connect Internet


- Update all packages

    apt-get update
    apt-get install iputils-ping curl net-tools
    apt-get install iproute2


- Check if the commands

    which ping
    which curl
    which ifconfig
    which ip


- Ping

    curl -I https://www.google.com


- Install Neovim//Nano ..

    apt-get install neovim
